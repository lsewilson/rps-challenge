# Rock, Paper, Scissors Challenge


## Description

This repository is a simple web application where a user can play rock, paper, scissors
against a computer player.

## Set Up Instructions
```
$ git clone https://github.com/lsewilson/rps-challenge.git
$ cd rps-challenge
$ bundle
$ rackup
```
